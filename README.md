Touchscreen Stuff
=================

Building a Touchscreen from components.

Pinouts
-------

Computer Side

| USB B receptacle | 3M Microtouch USB controller |
|------------------|------------------------------|
| 1                | +5vdc                        |
| 2                | data (DN)                    |
| 3                | data (DP)                    |
| 4                | power return                 |
| 5                | shield                       |

Touchscreen Side

| Transducer | fcn | Cat 5 Ethernet |
| ---------- | --- | -------------- |
| 1          | ul  | 2              |
| 2          | ur  | 4              |
| 3          | gnd | 1, 3, 5, 7     |
| 4          | lr  | 6              |
| 5          | ll  | 8              |
